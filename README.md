# Community Website
Static site with 5 pages. Tech: HTML5, Bootstrap 5, vanilla JS. Host: GitHub Pages/Netlify.

## Local Preview
Open `index.html` in a browser or use a local server:
- VS Code Live Server extension, or
- `python3 -m http.server` (then visit http://localhost:8000)

## Deploy
- GitHub Pages: push to `main`, enable Pages in repository settings.
- Netlify: drag-and-drop the folder to app.netlify.com.

## Content
- Update images in `assets/images/`
- Update tables in `events.html` (or load from `data/events.csv`)
- Add articles inside `/articles`
